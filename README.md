# 基于po模型开发

包目录说明
- base包：负责存放公用的操作方法
- page包：负责用于元素的定位和数据的维护
- handle包:获取元素后的操作
- business包:负责业务逻辑用例
- listener包:报告生成监听器
- utils包:工具类
